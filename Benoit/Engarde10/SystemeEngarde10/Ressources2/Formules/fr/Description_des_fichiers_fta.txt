Les fichiers .fta de description de formules sont chargés lorsque l’on charge une formule. Ces fichiers se trouvent :
- dans le répertoire Ressources2\Formules\fr pour les formules fournies décrites en français
- dans le répertoire Ressources2\Formules\en pour les formules fournies décrites en anglais
- dans le répertoire Ressources1\mesFormules pour les formules personnelles

Il est possible d’enregistrer comme modèle la formule en cours d’une compétition. Elle est enregistrée dans le répertoire Ressources1\mesFormules

Les fichiers .fta sont éditables avec un éditeur de texte.  Leur modification est délicate. Il faut bien tester les modifications sur des simulations et lorsque les suite_tableaux ou les description_tableau ont été modifiés, vérifier que le classement général d’une compétition simulée est correct.

Les fichiers .fta de description de formules peuvent contenir quatre sortes d'entités.
Toutes les entités ont une « classe » qui doit être l’une des quatre sortes  listées ci-dessous, exemple :  [classe suite_tableaux]
Toutes les entités ont une  « cle » qui doit être un entier. Toutes les « cle » d’une même « classe » doivent être différentes.

Les lignes commençant par un point-virgule sont des commentaires ignorés au chargement. Les lignes de ce type au début du fichier sont affichées lorsque l’on veut choisir une formule (maximum trois).

1) Les suite_tableaux sont obligatoires et visibles dans l’onglet « Formule » d’Engarde

2) Les description_tableau sont facultatives, elles permettent d’attribuer des noms précis aux tableaux. Quand elles sont absentes, les description_tableau sont construites par Engarde. Elles sont visibles dans l’onglet « Formule » d’Engarde

3) les multi_affichage permettent de placer plusieurs tableaux dans un même document. Leur utilisation se fait avec le bouton « Document » de l’onglet « Tableaux » et l’item « Ces tableaux ». Ces entités ne sont pas visibles dans Engarde. Il est seulement possible de les ajouter ou de les modifier dans les fichiers .fta Exemples :
[liste_tab_a_imp ( ((a256 a64)) )]   pour un document affichant du tableau a256 au tableau a64
[liste_tab_a_imp ( ((b8 b8)) ((c8 c8)) ((d8 d2)))]  pour un document affichant le tableau b8 puis le tableau c8 puis du tableau d8 au tableau d2
[liste_tab_a_imp ( ((c4 c2 d2 d2)) )] pour un document affichant du tableau c4 au tableau c2 et à sa gauche du tableau d2 au tableau d2 (les battus de c4 sont en d2)
[liste_tab_a_imp ( ((a16 a2 b8 b2) (h2 e2)) )] pour un document affichant du tableau a16 au tableau a2 et à sa gauche du tableau b8 au tableau b2 (les battus de a16 sont en b8) puis en dessous à droite le tableau h2 (les battus du tableau a4) et à gauche le tableau e2 (les battus du tableau b4). Mettre () à la place de h2 si l'on ne veut pas h2 et que l'on veut e2
[liste_tab_a_imp ( ((a16 a2 b8 b2) (h2 e2)) ((c4 c2 d2 d2)) )] pour mettre ensemble les tableaux des deux précédentes descriptions. 

4) les fichier_web permettent de placer plusieurs tableaux dans un même fichier web. Leur utilisation se fait avec le bouton « Web » de l’onglet « Tableaux » et l’item « Publier les tableaux sur le web ». Ces entités sont visibles et modifiables dans Engarde avec le bouton « Web » de l’onglet « Tableaux » et l’item « Définit les fichier web de tableaux ».
