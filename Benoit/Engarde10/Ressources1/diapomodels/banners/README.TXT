Customization of the transition screen of DiapoEngarde

The transition screen of DiapoEngarde, also called Logo, can have a banner on the top and on the bottom.

The top banner must be a JPEG file with the name TopScreenTransition.jpg
The bottom banner must be a JPEG file with the name BottomScreenTransition.jpg
 
These two files must be put in the directory Ressources1\diapomodels\banners

It is possible to have only one of them.

With a screen having width = 1280 pixels:
- For the top banner, the recommended width is 1280 and the recommended height is 409
- For the bottom banner, the recommended width is 1280 and the recommended height is 204

The optimal dimensions are displayed in DiapoEngarde by the menu "Logo/Taille des banničres" of the main window.

An example is provided in the directory Ressources1\diapomodels\banners\example
